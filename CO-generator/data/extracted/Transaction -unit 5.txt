4.Phantom Read Problem 
 • It occurs when a transaction reads a variable 
once from the buffer and when it tries to read 
that same variable again, an error occurs saying 
that the variable does not exist.  
– T1 reads X. 
– T2 reads X. 
– T1 deletes X. 
– T2 tries reading X but does not find it.
